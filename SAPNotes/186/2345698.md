2345698 - Missing Authorization check in FI-AA-AA

**SAP Security Note 2345698 - Missing Authorization Check in FI-AA-AA**

**Header Data**
- **SAP Note/KBA:** 2345698
- **Type:** SAP Security Note
- **Version:** 1
- **Recency:** New
- **Language:** English
- **Master Language:** English
- **Category:** Program error
- **Priority:** Correction with medium priority
- **Status:** Released for Customer
- **Released On:** 01.03.2017
- **Component:** Financial Accounting > Asset Accounting > Basic Functions

**Description**

### Symptom
FI-AA-AA does not perform necessary authorization checks for an authenticated user, resulting in escalation of privileges.

Some well-known impacts of Missing Authorization checks are:
- Abuse functionality restricted to a particular user group
- Read restricted data

### Other Terms
Access control, Authorization error, Authorization profile

### Solution
Implement Support Package or correction instructions.

The correction implements additional checks to restrict the data to be read.

3256571 - [CVE-2022-41214] Multiple vulnerabilities in SAP NetWeaver Application Server ABAP and ABAP Platform

---

### Description

SAP has released Security Note **3256571** addressing multiple vulnerabilities in the SAP NetWeaver Application Server ABAP and ABAP Platform.

#### Vulnerabilities:

1. **CVE-2022-41214: Directory/Path Traversal**
   - **Description:** Insufficient input validation allows an attacker with high-level privileges to delete a restricted file via a remote-enabled function. This can compromise the integrity and availability of the application.
   - **CVSS Score:** 8.7 
   - **CVSS Vector:** [CVSS:3.0/AV:N/AC:L/PR:H/UI:N/S:C/C:N/I:H/A:H](https://nvd.nist.gov/vuln-metrics/cvss/v3-calculator?vector=CVSS:3.0/AV:N/AC:L/PR:H/UI:N/S:C/C:N/I:H/A:H)
   - **CVE Link:** [CVE-2022-41214](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2022-41214)

2. **CVE-2022-41212: Directory/Path Traversal**
   - **Description:** Insufficient input validation allows an attacker with high-level privileges to read a restricted file via a remote-enabled function, compromising confidentiality.
   - **CVSS Score:** 4.9
   - **CVSS Vector:** [CVSS:3.0/AV:N/AC:L/PR:H/UI:N/S:U/C:H/I:N/A:N](https://nvd.nist.gov/vuln-metrics/cvss/v3-calculator?vector=CVSS:3.0/AV:N/AC:L/PR:H/UI:N/S:U/C:H/I:N/A:N)
   - **CVE Link:** [CVE-2022-41212](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2022-41212)

#### Solution

Implement the support package mentioned in [SAP Note 3256571](https://me.sap.com/notes/3256571) or follow the respective correction instructions provided in the note.

#### Affected Software Components

- **SAP_BASIS** versions 700 to 757, 789, 790, 804.

For detailed information about affected versions and support packages, refer to the [SAP Note 3256571](https://me.sap.com/notes/3256571).

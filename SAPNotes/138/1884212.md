**SAP Security Note 1884212: Bank Statement - Potential Directory Traversal**

**Description**

A vulnerability has been identified in the FI-BL-PT-BA component that allows an attacker to perform directory traversal. This can enable the attacker to write arbitrary files to the remote server, potentially leading to data corruption or alteration of system behavior.

**Symptom**

An attacker can exploit the directory traversal vulnerability in FI-BL-PT-BA to overwrite data on the remote system.

**Reason and Prerequisites**

The FI-BL-PT-BA component fails to correctly validate the file path for user-submitted files. This lack of validation permits attackers to overwrite data on the remote system.

**Solution**

Implement SAP Security Note [1884212](https://me.sap.com/notes/0001884212/E) to address the vulnerability.

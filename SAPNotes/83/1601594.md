SAP Security Note 1601594 - Potential disclosure and modification of persisted data in CRM-IM-IPM-BTX

```markdown
**SAP Security Note:** 1601594  
**Title:** Potential disclosure and modification of persisted data in CRM-IM-IPM-BTX  
**Type:** SAP Security Note  
**Priority:** Correction with high priority  
**Status:** Released for Customer  
**Released On:** 08.11.2011  
**Language:** English  

**Affected Component:**  
Customer Relationship Management > Media > Intellectual Property Management > Business Transactions Licensing (CRM-IM-IPM-BTX)

### Description

**Symptom:**  
A malicious user can exploit the IPM Mass Change template UI using specially crafted inputs to execute arbitrary database commands to retrieve, modify, or remove data persisted by the system.

**Other Terms:**  
IPM Mass Change, Mass Change Template, SQL injection, database

**Reason and Prerequisites:**  
The problem is caused by an SQL injection vulnerability. The code composes an SQL statement including strings that can be altered by a malicious user. The manipulated SQL statement can then be used to retrieve additional information from the database or to potentially modify it.

**Solution:**  
Please implement the attached correction instructions. This fix is valid for CRM 7.01.
```

SAP Security Note 3203065 - CVE-2022-31589: Segregation of Duty Vulnerability in IL FI-AP File from SHAAM Program

```markdown
**SAP Security Note 3203065 - [CVE-2022-31589](https://cve.mitre.org/cgi-bin/cvekey.cgi?keyword=CVE-2022-31589): Segregation of Duty Vulnerability in IL FI-AP File from SHAAM Program**

**Symptom**
Due to an improper authorization check, business users utilizing the Israeli File from the SHAAM program (`/ATL/VQ23` transaction) are granted excessive authorizations. This may allow users to access data that should remain restricted.

**Reason and Prerequisites**
- The `F_BKPF_BUK` authorization object was previously checked for both classical and extended withholding tax.
- The issue affects the Israeli Localization in SAP.

**Solution**
Implementing this SAP Note will restrict the `F_BKPF_BUK` authorization check solely to companies using classical withholding tax. To resolve the vulnerability:
- **Manual Correction**: Remove the `F_BKPF_BUK` authorization object from the profiles of users who do not require it.
- **Support Package**: Alternatively, apply the relevant Support Package provided in the note.

**Affected Components**
- **Financial Accounting (FI) Components**:
  - SAP_FIN (Versions 618, 730)
  - S4CORE (Versions 100 to 108)
  - C-CEE (Versions 110_600, 110_602, 110_603, 110_604, 110_700, 110_720)
```

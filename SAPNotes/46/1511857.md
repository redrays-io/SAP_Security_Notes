SAP Security Note 1511857 - Potential modification of persisted data in BC-I18

**Overview**
SAP has released Security Note **1511857**, addressing a critical vulnerability in the **Basis Components > Internationalization (I18N) (BC-I18)** module. This vulnerability allows malicious users to exploit specially crafted inputs to modify database commands, which can lead to unauthorized modification of persisted data within the system.

**Details**
- **Type:** SAP Security Note
- **Category:** Program error
- **Priority:** Correction with high priority
- **Status:** Released for Customer
- **Released On:** December 14, 2010
- **Component:** BC-I18 (Basis Components > Internationalization)
- **Affected Versions:** SAP_BASIS 610 to 640, 700 to 702, 710 to 730, 72L to 800

**Description**
A malicious user can exploit BC-I18 by using specially crafted inputs to modify database commands. This SQL injection vulnerability allows the alteration of data persisted by the system, potentially compromising data integrity and system security.

**Solution**
To mitigate this vulnerability, apply the attached correction instructions provided by SAP. These instructions are crucial for securing your SAP environment against potential exploits targeting the BC-I18 component.

**SAP Security Note 1553868 - Potential Modification or Disclosure of Persistent Data in FS-BA**

**Symptom**

A malicious user can exploit FS-BA-AN-FSP and use specially crafted inputs to modify database commands. This results in either the retrieval of additional information or the modification of data persisted by the system.

**Other Terms**

- SQL injection
- Database
- FS-BA-AN-FSP

**Reason and Prerequisites**

The problem is caused by an SQL injection vulnerability. The code composes an SQL statement that contains strings that can be altered by a malicious user. The manipulated SQL statement can then be used to retrieve additional data from the database or to modify the data.

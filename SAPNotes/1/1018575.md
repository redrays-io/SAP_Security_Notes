**Title:** Cross-site scripting (XSS) using the IGS

**Type:** SAP Security Note  
**Version:** 2  
**Recency:** New  
**Language:** English  
**Priority:** Correction with high priority  
**Status:** Released for Customer  
**Released On:** 08.10.2009  
**Component:** Basis Components > Frontend Services (SAP Note 1322184) > Internet Graphics Server

### Symptom
Cross-site scripting (XSS)

### Other Terms
XSS, cross-site scripting, security update

### Reason and Prerequisites
You have defined an HTTP listener and have specified HTTP administration access to your Internet Graphics Server (IGS). The content of HTTP parameters is returned without further checking.

### Solution
Please install IGS 6.40 patch 18 or IGS 7.00 patch 8.

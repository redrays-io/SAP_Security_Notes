1027100 - CST Patch Collection 08 2007

**Title:** 1027100 - CST Patch Collection 08 2007

**Description:**
This SAP Security Note addresses various errors in the Client Server Technology (CST) components of SAP BASIS. The patch collection resolves multiple issues across different releases, ensuring improved stability and performance of the CST environment.

**Symptom:**
Errors encountered in the Client Server Technology area, including but not limited to:
- Dispatcher issues
- Task handler malfunctions
- Gateway and message server errors
- Problems with standalone enqueue server, ICM, and SAP Web Dispatcher

**Other Terms:**
Components affected include:
- Cpiclib
- disp+work
- enqt
- enrepserver
- gwrd
- icman
- icmon
- jstart
- lgtst
- msg_server
- sapwebdisp
- rfclib

**Reason and Prerequisites:**
The issues are caused by various errors in critical CST components such as the dispatcher, task handler, gateway, message server, enqueue server, ICM, and SAP Web Dispatcher.

**Solution:**
The patch collection provided in this note resolves the following errors:

**Release 4.6D:**
- **1020281:** Trace message: sofcancel unblocked

**Release 6.40:**
- **1025489:** JCMon may consume 100% CPU
- **1025488:** JLaunch may crash during shutdown
- **1021425:** Message server HTTP redirect changes URL query string
- **1020281:** Trace message: sofcancel unblocked
- **1019719:** Trace: "NiPGetPort: getsockname (22: Invalid argument)"
- **1018968:** Enqueue replication server does not terminate after failover
- **978078:** Maximum number of gateways reached
- **698017:** ICM Patch Collection (6.40)

**Release 7.00:**
- **1025489:** JCMon may consume 100% CPU
- **1025488:** JLaunch may crash during shutdown
- **1021425:** Message server HTTP redirect changes URL query string
- **1021205:** Repeated debugging in SM50 does not work
- **1020281:** Trace message: sofcancel unblocked
- **1018968:** Enqueue replication server does not terminate after failover
- **1018109:** Meaningless characters in table names of statistic
- **978078:** Maximum number of gateways reached
- **974284:** SAP Web Dispatcher 7.00: Patch history
- **851852:** ICM Patch Collection (7.00)

**Patch Levels Required:**
- **4.6D:** 2307
- **6.40:** 171
- **7.00:** 98

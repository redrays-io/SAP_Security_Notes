**SAP Note 2490047 - Switchable Authorization Checks for RFC in Customer Master Data**

**Overview:**
This SAP Security Note introduces new switchable authorization checks for RFC function modules in customer master data to enhance system security.

**Details:**
- **Number:** 2490047
- **Type:** SAP Security Note
- **Version:** 2
- **Language:** English
- **Category:** Program error
- **Priority:** Correction with medium priority
- **Status:** Released for Customer
- **Released On:** 08.01.2019
- **Component:** Logistics - General > Logistics Basic Data > Business Partners > Customer Master

**Description:**
Remote Function Call (RFC) function modules are protected by the authorization object `S_RFC` by default. However, for certain RFC function modules, `S_RFC` checks may not be sufficient. This note implements new switchable authorization checks that can be activated to provide enhanced security for these function modules.

**Affected RFC Function Modules:**
- `BAPI_CUSTOMER_SEARCH`
- `BAPI_CUSTOMER_SEARCH1`
- `SDCA_CONTACT_VALUE_REQUEST`

**Solution:**
1. **Implementation:**
   - The new authorization checks are delivered as inactive to maintain compatibility.
   - They can be activated using transaction `SACF` following the manual correction instructions.

2. **Manual Correction Instructions:**
   - **Download the Scenario File:** [SACF_LO_MD_CUSTOMER_VENDOR_F4H.TXT](https://userapps.support.sap.com/sap/support/sapnotes/public/services/attachment.htm?iv_key=002075125800001643912017&iv_version=0002&iv_guid=6EAE8B27FE511ED795BCDBEEDFE440C6)
   - **Activate the Scenario:**
     - Start transaction `SACF_TRANSFER`.
     - Select the **Upload** option.
     - Check the **Work Area Scenario Definition** box.
     - Enter the Scenario Name: `LO_MD_CUSTOMER_VENDOR_F4H`.
     - Choose **Replace existing scenarios** and ensure **Test Mode** is unchecked.
     - Execute and upload the scenario file.

**Prerequisites:**
- **SAP Note 2373735:** Must be implemented for releases starting with ECC 2005 EhP 616 as it provides the required SACF scenario.

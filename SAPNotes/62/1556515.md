**SAP Security Note 1556515 - SLL-LEG-FUN-UPL: Directory Traversal**

**Overview**
SAP has released Security Note **1556515**, addressing a potential **Directory Traversal** vulnerability in the **SLL-LEG-FUN-UPL** component. This vulnerability allows malicious users to read or write arbitrary files on the remote server, potentially leading to the disclosure of confidential information or corruption of data.

**Symptom**
- Potential Directory Traversal in the following component:
  - **SLL-LEG-FUN-UPL**

**Affected Software Components**
- **SLL-LEG** versions: 710, 720, 800, 900

**Reason and Prerequisites**
1. The programs in the correction instructions contain vulnerabilities that allow malicious users to read arbitrary files on the remote server.
2. Some programs allow malicious users to write arbitrary files on the remote server, potentially corrupting data or altering system behavior.

**Solution**
- **Prerequisite:** Apply Security Note [1497003](https://me.sap.com/notes/1497003) which addresses additional vulnerabilities and is required before implementing this note.
- Implement the corrections provided in this note to mitigate the directory traversal vulnerabilities.

**Correction Instructions**
- 8 correction instructions are provided to address the vulnerabilities in the **SLL-LEG** component.

**Manual Activities**
- **1 Manual Activity:** Creation of a system message in the message class `/SAPSLL/CON_LDET`.

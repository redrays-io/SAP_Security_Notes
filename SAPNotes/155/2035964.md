SAP Security Note 2035964 - Potential Information Disclosure in NFM from TPS Systems, Inc.

**Overview**
SAP Security Note [2035964](https://me.sap.com/notes/2035964) addresses a potential information disclosure vulnerability in the Network File Manager (NFM) provided by TPS Systems, Inc., specifically affecting the IS-R-TRN-TFT (Transnet Trickle File Transfer) component.

**Symptom**
An attacker can discover sensitive information related to NFM from TPS Systems, Inc. used by IS-R-TRN-TFT. This information may be utilized to tailor attacks against NFM and IS-R-TRN-TFT, potentially compromising system security.

**Details**
- **Affected Component:** Industry-Specific Components > Industry-Specific Component Retail > Transnet > Transnet Trickle File Transfer (IS-R-TRN-TFT)
- **CVE Identifier:** [CVE-2014-0224](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0224)
- **Release Date:** August 12, 2014
- **Priority:** High
- **Impact:** Low risk if NFM is used within a VPN or private network. Higher risk if exposed to external networks.

**Reason and Prerequisites**
Information such as transaction and file data can be accessed using IS-R-TRN-TFT, which may aid attackers in targeting NFM and related components more effectively.

**Solution**
Upgrade NFM to version **2.5.4.17** or higher to mitigate the SSL/TLS MITM vulnerability (CVE-2014-0224). The update addresses the vulnerability by patching the affected versions that use OpenSSL.

The installers for both NFM Server and NFM Client version 2.5.4.17 are available as zipped attachments in the SAP Security Note. Ensure all parts of the multipart zip files are downloaded to the same folder before unzipping. Refer to the included `NFM_Readme.txt` for detailed installation steps.

**SAP Security Note 1589919 - Hard-coded credentials in Function Builder**

**Description**

A malicious user can be authenticated to Function Builder without having their own legitimate credentials, or they may escalate privileges.

**Impact**

- **Symptom:** Unauthorized access to Function Builder, allowing potential privilege escalation.
- **CVSS Base Score:** 3.5
- **CVSS Vector:** AV:N/AC:M/AU:S/C:N/I:P/A:N

**Reason and Prerequisites**

The vulnerability is caused by an option to suppress authorization checks against developer permissions in the program's source code. A malicious user can log on to the system and create ABAP function modules without having been assigned legitimate access by the system administrator. If a user already has privileges to log on, an escalation of privileges may be possible.

**Solution**

Please apply the support package mentioned in this note at least, or follow the respective correction instructions.

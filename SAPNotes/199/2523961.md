### SAP Security Note 2523961 - [CVE-2018-2360] Missing Authentication check in Startup Service

**Type**: SAP Security Note  
**Version**: 7  
**Recency**: New  
**Language**: English  
**Category**: Program error  
**Priority**: Correction with medium priority  
**Release Status**: Released for Customer  
**Released On**: 09.01.2018  
**Component**: Basis Components > Client/Server Technology > Startup Service  

#### **Description**

**Symptom**  
SAP Startup Service is missing an authentication check for functionalities that require user identity and cause consumption of file system storage.

**CVSS Information**  
**CVSS v3 Base Score**: 5.8 / 10  
**CVSS v3 Base Vector**:

| Availability Impact (A)                | Low (L)         |

*Note*: The CVSS score provided is an estimate and does not account for specific system configurations or environments.

**Other Terms**  
Broken authentication, sapstartsrv, [CVE-2018-2360](http://cve.mitre.org/cgi-bin/cvename.cgi?name=2018-2360)

#### **Solution**

- Obsolete coding has been deactivated.
- Implement the patches referenced by this SAP Note.

### SAP Security Note 2516864 - Missing Authorization Check in QM

**Release Date:** February 2, 2018  
**Component:** Quality Management (QM)  
**Priority:** Correction with medium priority  
**Status:** Released for Customer  
**Language:** English (Master Language: German)  
**Category:** Program error

#### **Symptom**
The Quality Management (QM) module does not perform the required authorization checks for an authenticated user, which may result in privilege escalation. The known effects of the missing authorization check include:
- Inappropriate usage of functions restricted to certain user groups.
- Reading, modification, or deletion of data to which access should be restricted.

#### **Reason and Prerequisites**
Insufficient authorization checks have led to this vulnerability.

#### **Solution**
Implement the correction instructions provided in this SAP Note to address the missing authorization checks.

#### **Correction Instructions**
- [SAP_APPL Corrections](https://me.sap.com/corrins/0002516864/1)
- [S4CORE Corrections](https://me.sap.com/corrins/0002516864/19773)

#### **Valid For:**
This SAP Note is applicable to the following software component versions:
- **SAP_APPL:** 46C, 470, 500, 600, 602, 603, 604, 605, 606, 616, 617, 618
- **SAPSCORE:** 110
- **S4CORE:** 100, 101, 102

#### **Side Effects**
This document is related to the following side effect:
- [SAP Note 3104603: Missing link of material documents to inspection lots of origin 17 due to missing authorization](https://me.sap.com/notes/0003104603)

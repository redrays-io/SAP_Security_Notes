2915126 - [CVE-2020-6260] Incomplete XML Validation in SAP Solution Manager (Trace Analysis)

**Description**

**SAP Note/KBA**: 2915126  
**Type**: SAP Security Note  
**Version**: 12  
**Recency**: New  
**Language**: English  
**Category**: Program error  
**Priority**: Correction with medium priority  
**Status**: Released for Customer  
**Released On**: 09.06.2020  
**Component**: Service > SAP Solution Manager > Diagnostics > Applications > Trace Analysis

**Symptom**

This note addresses two vulnerabilities caused by incomplete XML validation:

1. **CVE-2020-6260**
   - **Description**: SAP Solution Manager (Trace Analysis) allows an attacker to inject superfluous data that can be displayed by the application. The application shows additional data that do not actually exist.
   - **CVSS**: 6.5  
   - **CVSS Vector**: CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:L

2. **CVE-2020-6261**
   - **Description**: Allows log injection into the trace file, impairing the readability of the trace file.
   - **CVSS**: 5.3  
   - **CVSS Vector**: CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:N

**Reason and Prerequisites**

Incomplete XML validation in SAP Solution Manager leads to the mentioned vulnerabilities.

**Solution**

The application will no longer allow unauthorized data to be written into the trace file. Apply the latest LM-SERVICE patch as detailed below.

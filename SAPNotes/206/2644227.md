2644227 - [CVE-2018-2437] Unauthorized Command Execution in SAP Internet Graphics Server (IGS)

```markdown
## Symptom

The SAP Internet Graphics Service (IGS) allows an attacker to externally trigger IGS command executions through the multiplexer RFC listener, which can lead to:

- Disclosure of information
- Malicious file insertion or modification

## Reason and Prerequisites

Access to SAP Internet Graphics Server (IGS) is not sufficiently restricted.

## Solution

Please consult and review [SAP Note 1425765](https://me.sap.com/notes/1425765), specifically the section related to the configuration of the `reginfo` file. Create a `reginfo` entry for the SAP Internet Graphics Server (IGS) with the following arguments:

```
P TP=IGS.<SID> HOST=local CANCEL=local ACCESS=local
```
**or**

```
P TP=IGS.<SID> HOST=local CANCEL=local ACCESS=internal
```

For a production environment, we strongly recommend that the `ACCESS` condition be restricted to either "local" or "internal". For more information on how to configure your `reginfo` file, please review [SAP Note 1425765](https://me.sap.com/notes/1425765) carefully.

This SAP Note provides consulting. For general security reasons, please always use the latest SAP Internet Graphics Server (IGS) version available.
```

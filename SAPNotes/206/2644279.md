**SAP Security Note 2644279 - [CVE-2018-2462] Missing XML Validation Vulnerability in BEx Web Java Runtime Export Web Service**

**Description:**
A vulnerability has been identified in the BEx Web Java Runtime Export Web Service where XML documents accepted from untrusted sources are not sufficiently validated. This issue could allow attackers to perform XML-related attacks, such as DTD attacks, potentially leading to unauthorized access or data manipulation.

**CVSS v3 Base Score:** 8.8 / 10  
**CVSS v3 Vector:**  
- **Attack Vector (AV):** Network (N)  
- **Attack Complexity (AC):** Low (L)  
- **Privileges Required (PR):** Low (L)  
- **User Interaction (UI):** None (N)  
- **Scope (S):** Unchanged (U)  
- **Confidentiality Impact (C):** High (H)  
- **Integrity Impact (I):** High (H)  
- **Availability Impact (A):** High (H)  

**Impact:**
An attacker could exploit this vulnerability to manipulate XML data, leading to high impacts on confidentiality, integrity, and availability of the affected systems.

**Solution:**
To address this vulnerability, an XML parser update has been implemented to enhance validation against XML-related attacks. Administrators should import the relevant BI Java Patch for their specific NetWeaver BI version.

- **Affected Versions:**  
  - SAP NetWeaver BI 7.30, 7.31, 7.40, 7.41

- **Actions Required:**  
  Import the appropriate BI Java Patch into your BI system. Refer to [SAP Note 2470973](https://me.sap.com/notes/2470973) for detailed delivery information and available patch options. Additionally, consult [SAP Note 1512355](https://me.sap.com/notes/1512355) for the patch delivery schedule.

For more comprehensive information, please refer to the [Generic Information Note 1506722](https://me.sap.com/notes/1506722).

SAP Security Note 1679401 - Unauthorized modification in BSP application CRM-MKT-MPL-CA

```markdown
**SAP Security Note 1679401** addresses a critical vulnerability in the BSP application **CRM-MKT-MPL-CA** that allows unauthorized modification of displayed application content. This issue can be exploited by malicious users to perform cross-site scripting (XSS) attacks, potentially obtaining authentication information from other legitimate users.

### Symptom
A malicious user can modify displayed application content without authorization and potentially steal authentication information, leading to user impersonation and compromised application security.

### Affected Components
- **CRM-MKT-MPL-CA**

### Reason
Certain HTML files within the **CRM-MKT-MPL-CA** component do not sufficiently encode output parameters, resulting in a cross-site scripting issue.

### Impact
- Unauthorized modification of application content
- Theft of user authentication information
- Potential impersonation of administrators, compromising application security

### Solution
Apply [SAP Note 1679401](https://me.sap.com/notes/1679401) or import the changes via the relevant support package.
```

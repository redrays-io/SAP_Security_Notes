SAP Security Note 3012021 - [Multiple CVEs] Multiple vulnerabilities in SAP Process Integration (Integration Builder Framework)

**Description:**

- **Unrestricted File Upload Vulnerability:**
  The Integration Builder Framework of SAP Process Integration allows an attacker with admin privileges to upload any file (including script files) without proper file format validation.

- **Missing XML Validation:**
  The application does not sufficiently validate an XML document accepted from a local source.

**Impact:**

- **Denial-of-Service (DoS):** Successful exploitation could lead to denial-of-service conditions.

**Affected Components:**

- **Component:** BC-XI-IBF-UI  
  **Path:** Basis Components > NetWeaver Process Integration (PI) > Framework > UI: Framework, Obj Lookup, Admin Pages, exchg prof

**CVEs Addressed:**

- [CVE-2021-27617](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2021-27617)
- [CVE-2021-27618](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2021-27618)

**Solution:**

Protective measures against XML Injection and Unrestricted File Type Upload have been applied at the code level. To safeguard your system, apply the [Support Packages and Patches](https://me.sap.com) referenced by this SAP Security Note.

**Support Package Patches:**

A comprehensive list of applicable support package patches can be found within the Security Note. Ensure that you apply the relevant patches for your specific software component versions.

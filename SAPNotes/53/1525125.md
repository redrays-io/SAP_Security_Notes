### Description

**Symptom**  
- **UPDATE 2: May 14, 2019:** With update 2 in [SAP Note 1408081](https://me.sap.com/notes/1408081/D), this SAP Note is obsolete.
- **UPDATE 1: November 3, 2010:** Correction of some typing errors.

*Changes not taken into account in the "Solution" section.*

**Other Terms**  
`reg_info`, `sec_info`, `SNC`, `Saprouter`, `DMZ`, `internal`, `local`, `gw/reg_no_conn_info`, `gw/acl_mode`

**Reason and Prerequisites**  
Spelling error in the original SAP Note; "internal" is correct, not "interal".

**Solution**  
The corrected minimum configuration is as follows:

```plaintext
#VERSION=2
P TP=* HOST=local
P TP=* HOST=internal CANCEL=internal ACCESS=internal
```

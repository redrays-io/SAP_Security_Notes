SAP Security Note 2266565 - SAPSSOEXT Process Crash During Ticket Verification

**Title:** 2266565 - SAPSSOEXT process crash during ticket verification

**Type:** SAP Security Note  
**Priority:** Correction with medium priority  
**Status:** Released for Customer  
**Released On:** 09.02.2016  
**Component:** Basis Components > Security - Read KBA 2985997 for subcomponents  
**Category:** Program error  

### Symptom
An attacker can remotely exploit SAPSSOEXT, potentially affecting the resources used.

**CVSS Information**  
- **Base Score:** 5.0 / 10  
- **Attack Vector:** Network (N)  
- **Access Complexity:** Low (L)  
- **Authentication:** None (N)  
- **Impact to Confidentiality:** None (N)  
- **Impact to Integrity:** None (N)  
- **Impact to Availability:** Partial (P)  

*Note: SAP provides this CVSS base score as an estimate of the risk. It does not consider your system configuration or operational environment. Conduct your own risk assessments when deciding on the applicability or priority of this security note.*

### Reason and Prerequisites
An attacker can trigger a condition causing the SAPSSOEXT process to enter an endless loop, consuming all available processing time and rendering the machine unresponsive. This can be exploited to launch a denial-of-service (DoS) attack.

**Affected Versions:**  
SAPSSOEXT versions prior to patch 15.

**Detection:**  
If SAPSSOEXT is used as a library in a non-SAP environment, check the version using the API method `MySapGetVersion`.

### Solution
Download patch 15 of SAPSSOEXT and replace your local installation.

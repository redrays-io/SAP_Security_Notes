SAP Security Note 3111110 - [CVE-2022-26100] Denial of Service (DOS) in SAPCAR

**Overview:**
SAPCAR has an input validation vulnerability that can lead to a Denial of Service (DoS) attack and potentially grant attackers privileged access to the system.

**Details:**
- **Vulnerability:** CVE-2022-26100
- **Type:** Buffer Overflow, Resource Exhaustion
- **Impact:** Local attackers with access to SAPCAR can exploit the vulnerability to crash the SAPCAR process, leading to DoS or elevated privileges.
- **CVSS Score:** 4.8  
  [CVSS Details](https://nvd.nist.gov/vuln/detail/CVE-2022-26100)

**Affected Components:**
- **Component:** BC-INS-TLS (Basis Components > Installation Tools > SAPCAR Tools)
- **Version:** 7.22

**Solution:**
Apply Support Package Patch **001115** for SAPCAR 7.22. You can download it from the [SAP Software Center](https://me.sap.com/softwarecenter/template/products/_APP=00200682500000001943&_EVENT=DISPHIER&HEADER=Y&FUNCTIONBAR=N&EVENT=TREE&NE=NAVIGATE&ENR=73555000100200014919&V=MAINT).

Alternatively, consider updating to a higher Support Package version as recommended in the SAP Note.

**Title**
3104456 - [CVE-2021-42062] Missing Authorization check in SAP ERP HCM

**Description**

### Symptom
SAP ERP HCM Portugal does not perform necessary authorization checks for a report that reads the payroll data of employees in a certain area. Although the affected report only reads payroll information, the attacker cannot modify any information or cause availability impacts.

### Other Terms
- **Access control**
- **Authorization error**
- **Authorization profile**
- [CVE-2021-42062](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2021-42062)

### Reason and Prerequisites
SAP ERP HCM lacks authorization checks to verify an authenticated user's permission to access certain functions. This omission may result in undesired system behavior.

### Solution
The affected functions now enforce proper access restrictions. Please implement the correction instructions provided in the note.

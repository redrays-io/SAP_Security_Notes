3144941 - [CVE-2022-26104] Missing Authorization check in SAP Financial Consolidation

**Symptom**

SAP Financial Consolidation does not perform necessary authorization checks for updating homepage messages, allowing an unauthorized user to alter the maintenance system message.

**Other Terms**

BFC, BOFC, access control, authorization error, authorization profile, [CVE-2022-26104](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2022-26104)

**Reason and Prerequisites**

Unauthorized access to SAP Financial Consolidation.

**Solution**

Modification of the homepage message requires the new functional role "Manage homepage message" to be assigned to the user that invokes the messages via the API.

1951071 - Missing authorization check in XX-CSC-CL-FI

**SAP Security Note 1951071**

**Type:** SAP Security Note  
**Version:** 2  
**Recency:** New  
**Language:** English  
**Category:** Program error  
**Priority:** Correction with medium priority  
**Status:** Released for Customer  
**Released On:** 15.03.2017  
**Component:** Miscellaneous > Country/Region-Specific Developments > Chile > use FI-LOC-FI-CL-REP (XX-CSC-CL-FI)

### Description

#### Symptom
An authenticated user can use functions of XX-CSC-CL-FI to which access should be restricted. This may result in an escalation of privileges.

#### Other Terms
Authorization, authorization check, XX-CSC-CL-FI.

#### Reason and Prerequisites
XX-CSC-CL-FI does not contain authorization checks for verifying an authenticated user's authorization to access some of its functions. This may result in undesired system behavior.

#### Solution
Apply the [correction instructions](https://me.sap.com/corrins/0001951071/15841) attached with the note.

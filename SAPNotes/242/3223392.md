3223392 - [CVE-2022-35292] Windows Unquoted Service Path Issue in SAP Business One

**Symptom**

In the SAP Business One application, when a service is created, the executable path contains spaces and isnâ€™t enclosed within quotes. This leads to a vulnerability known as **Unquoted Service Path**, which allows a user to gain SYSTEM privileges. If exploited by adversaries, it can be used to gain privileged permissions on a system or network, leading to a high impact on Confidentiality, Integrity, and Availability.

**Other Terms**
- Windows Unquoted Service Path
- Privilege Escalation
- [CVE-2022-35292](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2022-35292)

**Reason and Prerequisites**
The service path is not enclosed with quotes and includes spaces.

**Solution**
Customers need to upgrade to **SAP Business One FP2202HF1**.

*The patches can be found on the [SAP ONE Support Launchpad](https://launchpad.support.sap.com/#/softwarecenter).*

**SAP Security Note 3568307 - [CVE-2025-26657] Information Disclosure Vulnerability in SAP KMC WPC**

**Description**
---
**Symptom**
  
SAP KMC WPC allows an unauthenticated attacker to remotely retrieve usernames by a simple parameter query which could expose sensitive information, causing low impact on the confidentiality of the application. This has no effect on integrity and availability.

**Other Terms**
  
Information Exposure, Information Leak, [CVE-2025-26657](https://www.cve.org/CVERecord?id=CVE-2025-26657)

**Reason and Prerequisites**
  
Missing code check allows the attacker to retrieve remote SAP usernames by simple parameter query.

**Solution**
  
With this fix, the system checks whether the user is logged in and, if not, forces the user to log in.

Please implement the Support Packages and Patches referenced by this SAP Security Note.

**Attributes**
---
- **Externally Reported:** Yes

  - **Attack Complexity (AC):** Low (L)

  - **Scope (S):** Unchanged (U)

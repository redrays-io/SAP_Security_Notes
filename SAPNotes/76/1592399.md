1592399 - Directory traversal in data interface to SAP F&R

**Title:** 1592399 - Directory traversal in data interface to SAP F&R

**Description:**
A potential directory traversal vulnerability exists in the data interface to SAP Forecasting and Replenishment (F&R). This vulnerability allows a malicious user to write arbitrary files on the remote server, which could corrupt data or alter system behavior.

**Symptom:**
Potential directory traversal in the data interface to SAP F&R.

**Reason and Prerequisites:**
The program specified in the correction instructions contains a vulnerability that enables a malicious user to write arbitrary files on the remote server. This could lead to data corruption or changes in system behavior.

**Solution:**
The correction involves validating the logical file name to determine the physical file name. This ensures that only authorized file operations are permitted, eliminating the directory traversal vulnerability. These corrections are independent of Note 1497003.

SAP Security Note 2827086: Several security vulnerabilities in FRP 5.4.0 and FR Engine 5.4.0

```markdown
**SAP Security Note 2827086** addresses multiple security vulnerabilities affecting **SAP Forecasting and Replenishment for Retail (FRP 5.4.0)** and **FR Engine 5.4.0**. This includes vulnerabilities in the open-source software libraries utilized by these applications.

### **Symptom**

This security note identifies several vulnerabilities:

- **Memory Corruption:** Logical errors in memory management can lead to memory corruption, potentially causing system crashes or information disclosure.
- **Denial of Service (DoS):** Malicious actors can flood the service or crash the application, disrupting legitimate usage and impacting system availability.

**Impacts:**

- **Confidentiality, Integrity, Availability:** Vulnerabilities may compromise these key aspects, allowing attackers to gather information or disrupt services.
- **Further Attacks:** Information obtained from these vulnerabilities can be used to execute more severe attacks.

### **Reason and Prerequisites**

- **Memory Corruption:** Exploitation of vulnerabilities within the application can trigger memory corruption issues.
- **Denial of Service (DoS):** Crafted XML documents can cause excessive resource consumption, leading to application unresponsiveness.

### **Solution**

Apply the necessary patches to resolve these vulnerabilities:

- **FRP 5.4.0:** Refer to [SAP Note 2835532](https://me.sap.com/notes/2835532) for corrections (Patch 020).
- **FR Engine 5.4.0:** Refer to [SAP Note 2829766](https://me.sap.com/notes/2829766) for Release Note F&R (OTHER MERCH. SW) 2.2-Engines 5.4.0-Patch 22.

### **Affected Components**

- **FRP:** Version 5.4.0

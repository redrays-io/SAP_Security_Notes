SAP Security Note 2369541 - Missing XML Validation vulnerability in Enterprise Portal

```markdown
**SAP Security Note 2369541** - *Missing XML Validation vulnerability in Enterprise Portal*

**Type:** SAP Security Note  
**Version:** 3  
**Recency:** New  
**Language:** English  
**Category:** Program error  
**Priority:** Correction with medium priority  
**Status:** Released for Customer  
**Released On:** 14.02.2017  
**Component:** Enterprise Portal > SAP Enterprise Portal (On-Premise) > iViews > Upload content and actions

### Description

**Symptom**  
*Enterprise Portal* does not sufficiently validate an XML document accepted from an untrusted source.

Some well-known impacts of Missing XML Validation vulnerability are:
- Arbitrary files retrieval from the server
- Denial-of-service conditions in successful exploits

**CVSS Information**  
- **CVSS v3 Base Score:** 6.5 / 10  
- **CVSS v3 Base Vector:**
  - **Attack Vector (AV):** Network (N)
  - **Attack Complexity (AC):** Low (L)
  - **Privileges Required (PR):** Low (L)
  - **User Interaction (UI):** None (N)
  - **Scope (S):** Unchanged (U)
  - **Confidentiality Impact (C):** None (N)
  - **Integrity Impact (I):** None (N)
  - **Availability Impact (A):** High (H)

*SAP provides this CVSS v3 base score as an estimate of the risk posed by the issue reported in this note. This estimate does not take into account your own system configuration or operational environment.*

**Other Terms**  
XXE, XML input parsing

**Reason and Prerequisites**  
- **Prerequisites:** XML document is accepted from an untrusted source.
- **Reason:** XML validation is missing.

**Solution**  
Added validation to the accepted XML document.  
Under the "Support Packages & Patches" tab within this note, you can check for the appropriate SP & Patch level fixing this issue.

### Affected Software Components

- **EP-ADMIN 7.10 to 7.11**
- **EP-ADMIN 7.20**
- **EP-ADMIN 7.30**

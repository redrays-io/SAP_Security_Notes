### Title
**3331029 - [CVE-2023-33990] Denial of service (DOS) vulnerability in SAP SQL Anywhere**

### Description
**Symptom**  
SAP SQL Anywhere allows an attacker to prevent legitimate users from accessing the service by crashing the service. An attacker with a low-privileged account and access to the local system can write into the shared memory objects. This can be leveraged to perform a Denial of Service. Additionally, an attacker might be able to modify sensitive data in shared memory objects.

*Note*: This issue only affects SAP SQL Anywhere on Windows. Other platforms are not impacted.

**Other Terms**  
DoS, DDoS, Distributed Denial of Service, [CVE-2023-33990](https://www.cve.org/CVERecord?id=CVE-2023-33990)

### Solution
This vulnerability is fixed in SAP SQL Anywhere **17.0 SP1 PL11** or later versions.

- **Download the fixed version**: [Download for SNOTE](https://notesdownloads.sap.com/note/0040000000826162023)
- **PDF Version**: [Print PDF](https://me.sap.com/sap/support/sfm/notes/print/0003331029?language=en-US&token=5A80EA8A2A7A1070EDBDFB946EB91E55)

Install the appropriate fixed product version for your production environment from [SAP SQL Anywhere 17.0](https://launchpad.support.sap.com/#/softwarecenter/template/products/_APP=00200682500000001943&_EVENT=NEXT&HEADER=Y&FUNCTIONBAR=Y&EVENT=TREE&NE=NAVIGATE&ENR=67837800100800005169&V=MAINT&TA=ACTUAL/SYBASE%20SQL%20ANYWHERE).

*Note*: All releases subsequent to the published fixed releases will inherit the fixes described in this security note.

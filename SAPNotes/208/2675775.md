**SAP Note 2675775: Switchable Authorization checks for RFC in CRM Middleware**

**Description:**
This SAP Security Note introduces new switchable authorization checks for RFC function modules `SMOF0_GENERAL_DATA_HANDLER`, `SMOF0_INIT_DNL_START`, and `REFRESH_COMPARE_TABLES` in CRM Middleware. These checks are delivered inactive by default to ensure compatibility with existing processes and can be activated using transaction SACF.

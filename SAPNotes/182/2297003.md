2297003 - Missing Authorization check in EC-PCA-IS

**Component:** Enterprise Controlling > Profit Center Accounting > Information System (EC-PCA-IS)  
**Category:** Program error  

### Description

#### Symptom

EC-PCA-IS does not perform necessary authorization checks for an authenticated user, resulting in escalation of privileges.

Some well-known impacts of missing authorization checks are:

- Abuse functionality restricted to a particular user group
- Read restricted data

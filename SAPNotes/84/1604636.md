1604636 - Update #1 to Security Note 1511462

**Title:** 1604636 - Update #1 to Security Note 1511462

**Type:** SAP Security Note  
**Priority:** Correction with high priority  
**Released On:** October 12, 2011  
**Category:** Program error  
**Status:** Released for Customer

### Description

#### Symptom
A vulnerability allows a malicious user to trigger functionality in the SRM Java Tool Box without proper authentication and authorization. This update addresses the vulnerability outlined in Security Note 1511462 for the following Support Package Patch Level:
- **SP10 for SRM JAVATOOLBOX 7.0**

#### Other Terms
- Cross-Site Request Forgery (XSRF)
- SRM Java Tool Box
- Update Note

#### Reason and Prerequisites
Additional information for the Support Package Patch Level has been added to Security Note 1511462. The reference to "SRM 7.01 SP03" has been removed from the solution section of Note 1511462.

#### Solution
You can locate SRM Java Tool Box 7.0 in the Support Package and Patch Area of the [Service Marketplace](https://me.sap.com/).

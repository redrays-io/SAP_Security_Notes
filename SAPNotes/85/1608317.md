1608317 - Potential disclosure of persisted data in SAF

```markdown
**SAP Security Note 1608317 - Potential disclosure of persisted data in SAF**

**Release Date:** November 8, 2011  
**Type:** SAP Security Note  
**Priority:** Correction with high priority  
**Status:** Released for Customer

### Description

**Symptom:**  
An attacker can exploit the Software Agent Framework (SAF) using specially crafted inputs to modify database commands, resulting in the retrieval of additional information persisted by the system.

**Other Terms:**  
SQL injection, database, SAF, Software Agent Framework

**Reason and Prerequisites:**  
The issue is caused by an SQL injection vulnerability. The code constructs an SQL statement that includes strings that can be altered by an attacker. The manipulated SQL statement can be used to retrieve data from the database.

**Solution:**  
Implement this note or apply the related support package.

### Affected Software Components

- **SAP_ABA:** 700
- **CRMUIF:** 510, 520, 600
- **WEBCUIF:** 700, 701, 730, 731, 746
```

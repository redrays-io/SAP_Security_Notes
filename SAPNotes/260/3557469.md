SAP Security Note 3557469 - [CVE-2025-25245] Cross-Site Scripting (XSS) vulnerability in SAP BusinessObjects Business Intelligence Platform (Web Intelligence)

## Description

SAP BusinessObjects Business Intelligence Platform (Web Intelligence) contains a deprecated web application endpoint that is not properly secured. An attacker could exploit this by injecting a malicious URL in the data returned to the user. Successful exploitation could have a limited impact on confidentiality and integrity within the scope of the victim’s browser. There is no impact on availability.

**Other Terms:** XSS, DOM-based, [CVE-2025-25245](https://www.cve.org/CVERecord?id=CVE-2025-25245)

## Reason and Prerequisites

The input was not properly validated.

## Solution

The issue is fixed by removing the affected endpoint. Please apply the patches listed in the "Support Packages & Patches" section of this [SAP Security Note](https://me.sap.com/notes/3557469).

For Business Intelligence Platform maintenance schedule and strategy, see the Knowledge Base Article [2144559](https://me.sap.com/notes/2144559) in the References section.

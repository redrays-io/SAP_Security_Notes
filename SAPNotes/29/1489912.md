1489912 - BCA: Potential Directory Traversal

**SAP Security Note 1489912** addresses a potential directory traversal vulnerability in the following components:

- **IS-B-BCA**
- **FIN-FSCM-IHC**

### **Symptom**
A potential directory traversal vulnerability exists in the IS-B-BCA and FIN-FSCM-IHC components, allowing malicious users to write arbitrary files on the remote server. This could lead to data corruption or altered system behavior.

### **Affected Components**
- **Industry-Specific Components > Bank > Bank Current Accounts > Account Management (IS-B-BCA-AM)**
- **Financial Supply Chain Management > FSCM In-House Cash (FIN-FSCM-IHC)**

### **Reason and Prerequisites**
The vulnerability arises from a flaw in the program contained within the correction instructions. This flaw permits unauthorized file writes on the server, posing risks such as data corruption and system manipulation.

### **Solution**
Apply the provided correction or update to the relevant support package. Detailed instructions and support packages are available to address this vulnerability.

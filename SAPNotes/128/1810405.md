1810405 - Possible change/disclosure of persisted data in EH&S

```markdown
**SAP Note:** 1810405  
**Type:** SAP Security Note  
**Version:** 1  
**Language:** English  
**Release Status:** Released for Customer  
**Released On:** 24.09.2014  
**Component:** Environment, Health, and Safety / Product Compliance > Product Safety (EHS-SAF)  
**Priority:** Correction with medium priority  
**Category:** Program error  

### Description
An attacker can exploit EHS and use specially crafted inputs to modify database commands, resulting in the modification or display of data persisted by the system.

**Other Terms:**  
EHS31SP36PS, EHS32SP25PS, EHS2004SP29PS, EHS2005SP24PS, EHS62SP14PS, EHS63SP13PS, EHS64SP14PS, EHS65SP11PS, EHS66SP08PS, EHS616SP03PS, SQL injection, database, EH&S

### Reason and Prerequisites
The issue is caused by an SQL injection vulnerability. The code constructs an SQL statement using strings that can be manipulated by an attacker. This manipulated SQL statement can then be used to retrieve additional data from the database or modify existing data.

### Solution
Refer to the "Support Package" section below for information on which Support Packages contain the necessary corrections. Alternatively, you can implement the provided correction instructions.
```

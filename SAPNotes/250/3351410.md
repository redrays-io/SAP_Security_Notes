SAP Security Note 3351410 - [CVE-2023-36924] Log Injection vulnerability in SAP ERP Defense Forces and Public Security

## Overview

- **SAP Note/KBA**: 3351410
- **Type**: SAP Security Note
- **Version**: 9
- **Release Status**: Released for Customer
- **Released On**: 11.07.2023
- **Component**: IS-DFS-BIT-DIS
- **Category**: Program error
- **Priority**: Correction with medium priority

## Description

### Symptom

While using a specific function, SAP ERP Defense Forces and Public Security allows an authenticated attacker with admin privileges to write arbitrary data to the syslog file. On successful exploitation, an attacker could modify all the syslog data, causing a complete compromise of the application's integrity.

### Other Terms

- syslog
- SAP ERP Defense Forces and Public Security
- Log injection
- [CVE-2023-36924](https://www.cve.org/CVERecord?id=CVE-2023-36924)

### Reason and Prerequisites

While using specific functionality of SAP ERP Defense Forces and Public Security, sensitive information in syslog could be spoiled.

### Solution

The SAP ERP Defense Forces and Public Security specific functionality in question is retired and removed without replacement. Apply the Correction Instructions or upgrade to the corresponding Support Packages referenced in this Security Note.

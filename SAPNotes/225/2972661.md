SAP Security Note 2972661 - [CVE-2020-6367] Cross-Site Scripting (XSS) Vulnerability in SAP NetWeaver Composite Application Framework

**Symptom**
There is a Reflected cross site scripting vulnerability in SAP NetWeaver Composite Application Framework. An unauthenticated attacker can trick an unsuspecting authenticated user to click on a malicious link. The end userâ€™s browser has no way to know that the script should not be trusted, and will execute the script, resulting in sensitive information being disclosed or modified.

**Other Terms**
XSS, [CVE-2020-6367](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2020-6367), Reflected XSS, CSS

**Solution**
The URL parameters are now properly encoded to prevent a successful XSS attack. Implement the Support Packages and Patches referenced by this SAP Note.

**CVSS**
- **Score:** 8.2
- **Vector:** [CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:H/I:L/A:N](https://nvd.nist.gov/vuln-metrics/cvss/v3-calculator)

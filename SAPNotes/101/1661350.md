1661350 - ITS: Replace HTML encoding by new ABAP function

```markdown
**SAP Note Number:** 1661350  
**Type:** SAP Security Note  
**Priority:** Correction with high priority  
**Status:** Released for Customer  
**Released On:** 10.07.2012  
**Component:** BC-FES-ITS (SAP Internet Transaction Server)  

### Description
HTML encoding in the HTTP handler of ITS services will be replaced by a new ABAP function.

**Other Terms:** SAP GUI for HTML, WebGUI, IAC, EWT, ITS, service, encode, encoding, mask, escape, CL_HTTP_EXT_ITS

### Reason and Prerequisites
Update to the newest function.

### Solution
Please install the newest NetWeaver Basis Support Package!
```

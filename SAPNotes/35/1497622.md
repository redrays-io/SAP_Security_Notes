### Title
**1497622 - EC-EIS: Loading any source code using FM KXXC_DOWNLOAD**

### Description

#### Symptom
Due to an error in the EC-EIS Active Excel, it is possible for an attacker to execute any user-defined source code. This enables the attacker to gain control over the system and obtain secure increased privileges.

#### Other Terms
Backdoor, code upload

#### Reason and Prerequisites
There is a program error.

### **Title**
**1978131 - Missing authorization check in EP-PCT-PUR-BP**

---

### **Description**
#### **Symptom**
An authenticated user can use functions of EP-PCT-PUR-BP to which access should be restricted. This may result in an escalation of privileges.

#### **Other Terms**
Authorization, authorization check, EP-PCT-PUR-BP

#### **Reason and Prerequisites**
EP-PCT-PUR-BP does not contain authorization checks for checking an authenticated user's authorization to access some of its functions. This may result in undesired system behavior.

#### **Solution**
Import the relevant support package or implement this SAP Note using transaction [SNOTE](https://me.sap.com/sap/support/notes).

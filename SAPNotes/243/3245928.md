SAP Security Note 3245928 - Multiple CVEs in SAP 3D Visual Enterprise Viewer

```markdown
**SAP Security Note 3245928** addresses several vulnerabilities identified in **SAP 3D Visual Enterprise Viewer**. Below are the key details:

### **Symptom**
This security note covers multiple vulnerabilities that could be exploited when opening manipulated files from untrusted sources in SAP 3D Visual Enterprise Viewer. The vulnerabilities include:

1. **Arbitrary Code Execution**
   - **CVSS Score:** 7.0
   - **Vector:** CVSS:3.0/AV:L/AC:H/PR:N/UI:R/S:U/C:H/I:H/A:H
   - **CVE IDs:**
     - CVE-2022-41186 to CVE-2022-41202

2. **Denial of Service**
   - **CVSS Score:** 3.3
   - **Vector:** CVSS:3.0/AV:L/AC:L/PR:N/UI:R/S:U/C:N/I:N/A:L
   - **CVE IDs:**
     - CVE-2022-41188 to CVE-2022-41197

**Affected File Formats:**
- AutoCAD (.dwg, .dxf, TeighaTranslator.exe)
- Computer Graphics Metafile (.cgm, CgmCore.dll)
- Encapsulated Post Script (.eps, ai.x3d)
- EAAmiga Interchange File Format (.iff, 2d.x3d)
- Open Inventor File (.iv, vrml.x3d)
- Jupiter Tesselation (.jt, JTReader.x3d)
- Wavefront Object (.obj, ObjTranslator.exe)
- Right Hemisphere Binary (.rh, rh.x3d)
- Scalable Vector Graphics (.svg, svg.x3d)
- Visual Design Stream (.vds, vds.x3d)
- VRML Worlds (.wrl, vrml.x3d)
- SketchUp (.skp) [Support Removed]

### **Solution**
The issues are fixed in **Support Package SP015**. All 17 linked CVEs are addressed in this patch. To apply the fix:

- **Download for SNOTE:** [https://me.sap.com/notes/0003245928/Download](https://notesdownloads.sap.com/note/0040000001343252022)
- **PDF Version:** [https://me.sap.com/notes/0003245928/Print](https://userapps.support.sap.com/sap/support/sfm/notes/print/0003245928?language=en-US&token=B3EF4F1E7110909E2A9694390C3CB2CD)

**Patch Details:**
- **Support Package:** SP015
- **Component:** CA-VE-VEV
- **URL for Patch:** [https://me.sap.com/notes/0003245928/SP015](https://userapps.support.sap.com/sap/support/swdc/notes?cvnr=73555000100200017571&support_package=SP015&patch_level=000000)

For more information, refer to **SAP Note 3137595** on the release of SAP 3D Visual Enterprise Viewer 9.0 FP15: [https://me.sap.com/notes/3137595](https://me.sap.com/notes/3137595)

### **CVSS Details**
- **Confidentiality Impact:** High
- **Integrity Impact:** High
- **Availability Impact:** High

### **Attributes**

- **Externally Reported:** Yes
```

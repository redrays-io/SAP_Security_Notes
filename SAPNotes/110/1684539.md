1684539 - Missing authorization check in CA-BK (Bank)

```markdown
**SAP Security Note 1684539**

- **Type:** SAP Security Note
- **Released On:** June 12, 2012
- **CVSS Base Score:** 3.5 (AV:N/AC:M/AU:S/C:P/I:N/A:N)
- **Component:** CA-BK (Bank)
- **Priority:** Correction with high priority
- **Status:** Released for Customer

### Symptom

An authenticated user can use functions of CA-BK (Bank) to which access should be restricted. This may result in an escalation of privileges.

### CVSS Information

- **Base Score:** 3.5
- **Vector:** AV:N/AC:M/AU:S/C:P/I:N/A:N

*Note:* SAP provides this CVSS base score as an estimate of the risk posed by the issue reported in this note. This estimate does not consider your system configuration or operational environment. Conduct your own risk assessments when deciding on the applicability or priority of this SAP security note. More information can be found in the [FAQ section](https://me.sap.com/securitynotes/).

### Other Terms

Authorization, authorization check, CA-BK (Bank)

### Reason and Prerequisites

CA-BK (Bank) does not contain authorization checks for verifying an authenticated user's authorization to access some of its functions. This may lead to undesired system behavior.

### Solution

Apply the attached correction. You can [download the correction for SNOTE](https://notesdownloads.sap.com/note/0040000010006222017) or view the [PDF version](https://userapps.support.sap.com/sap/support/sfm/notes/print/0001684539?language=en-US&token=EEEA054B7D463527674A422192A04577).
```

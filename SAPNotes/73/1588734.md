### SAP Security Note 1588734 - FI-AP/AR: Potential Directory Traversal

**Release Date:** July 12, 2011  
**SAP Component:** Financial Accounting > Accounts Payable (FI-AP), Financial Accounting > Accounts Receivable (FI-AR)  
**Priority:** Correction with high priority  
**Status:** Released for Customer

#### **Description**
This security note addresses potential directory traversal vulnerabilities in the FI-AP and FI-AR components. These vulnerabilities may allow a malicious user to read or write arbitrary files on the remote server, potentially leading to the disclosure of confidential information or corruption of data.

**Symptoms:**
- Potential Directory Traversal in FI-AR and FI-AP components.

**Other Terms:**
- Path traversal

**Reason and Prerequisites:**
1. Vulnerabilities in the correction instructions enable reading arbitrary files, risking confidential data exposure.
2. Some programs allow writing arbitrary files, which can corrupt data or alter system behavior.

#### **Solution**
Refer to [Note 1497003](https://me.sap.com/notes/1497003) for additional information and instructions. Implementing the corrections from Note 1497003 is a prerequisite for applying this security note.

#### **Affected Software Components**
- SAP_APPL versions 31I, 40B, 45B, 46B, 46C, 470, 500, 600, 602, 603, 604, 605

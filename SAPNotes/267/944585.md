944585 - Security note: SAPDBA for Informix on UNIX platforms

```markdown
### Symptom
Unauthorized UNIX users can run commands with Informix rights at the Shell level.

### Other Terms
SAPDBA Informix security

### Reason and Prerequisites
You are using SAP with Informix on HP-UX, Solaris, AIX, TRUE64, or Linux. Due to an error in the SAPDBA, UNIX users can sometimes run commands with Informix rights with Shell authorization. The error occurs in all earlier SAPDBA versions prior to 700 and also in Version 700 with the patch number 100.

### Solution
To fix the error, import a SAPDBA Version 700 with a patch number 101 or higher. You can download this from [SAP me](https://me.sap.com/informix) â†’ sapdba, and you must also implement [Note 38401](https://me.sap.com/notes/38401).
```

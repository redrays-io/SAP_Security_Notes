# SAP Security Note 2668681 - CSRF Vulnerability in Manage Profit Centers

**Released On:** September 17, 2018  
**Type:** SAP Security Note  
**Version:** 4  
**Priority:** Correction with medium priority  
**Status:** Released for Customer  
**Component:** Controlling > Fiori UI for Overhead Cost Controlling (CO-FIO)  
**Language:** English

## Description

SAP Security Note **2668681** addresses a **Cross-Site Request Forgery (CSRF)** vulnerability in the **Manage Profit Centers** functionality. This vulnerability allows an attacker to trick an authenticated user into sending unintended requests to the web server due to insufficient CSRF protection.

### Impacts of CSRF Vulnerability:
- **Unauthorized Actions:** Attacker can perform actions on behalf of an authenticated user.
- **Loss of Non-repudiation:** Users cannot deny actions performed, leading to potential data integrity issues.

**Title:** [CVE-2023-24529] Cross-Site Scripting (XSS) Vulnerability in SAP NetWeaver AS ABAP (Business Server Pages Application)

**SAP Security Note:** [3282663](https://me.sap.com/notes/0003282663)

### **Symptom**
Due to lack of proper input validation, the BSP application (`CRM_BSP_FRAME`) allows malicious inputs from untrusted sources, which can be leveraged by an attacker to execute a reflected Cross-Site Scripting (XSS) attack. As a result, an attacker may be able to hijack a user session, read, and modify sensitive information.

### **CVSS Details**
- **Score:** 6.1
- **Vector:** [CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N](https://nvd.nist.gov/vuln-metrics/cvss/v3-calculator?vector=CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N)

### **Reason and Prerequisites**
This vulnerability occurs due to a programming error in the BSP application.

### **Solution**
1. **URL Encoding:** The URL parameters are now properly encoded to prevent successful XSS attacks.
2. **Apply Support Packages and Patches:** Implement the Support Packages and Patches referenced by this SAP Note.

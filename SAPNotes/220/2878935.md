SAP Security Note 2878935: XSS Vulnerability in SAP NetWeaver AS ABAP

**SAP Note Number:** 2878935  
**Title:** [CVE-2020-6246] Cross-Site Scripting (XSS) vulnerability in SAP NetWeaver AS ABAP (Business Server Pages Test Application SBSPEXT_TABLE)  
**Released On:** June 9, 2020  
**CVSS Score:** 6.1  

## Description

A **Cross-Site Scripting (XSS)** vulnerability has been identified in the **SAP NetWeaver AS ABAP â€“ Business Server Pages Test Application SBSPEXT_TABLE**. The application fails to sufficiently encode user-controlled inputs, which allows attackers to execute arbitrary scripts in the context of a user's browser session.

### Impact

Exploiting this vulnerability can lead to:

- **Defacing or modifying website content:** Attackers can alter or deface the displayed content.
- **Stealing user authentication information:** Sensitive data like session tokens can be compromised.
- **Impersonating users:** Attackers can access information and perform actions with the same privileges as the targeted user.

## Solution

To mitigate this vulnerability, SAP has provided patches that properly encode URL parameters to prevent XSS attacks. Administrators should:

1. **Download and implement the necessary Support Packages and Patches** referenced in [SAP Note 2878935](https://me.sap.com/supportpackage/SAPKB70038).
2. **Ensure the SICF node** `/sap/bc/bsp/sap/sbspext_table` is secured or disabled if not in use.

**SAP Security Note 1334907 - ICF, HTTPONLY flag for ICF cookies**

**Description**
This security note has been updated. For more detailed information, see [Security Note 1997862](https://me.sap.com/notes/1997862).

**Symptom**
You cannot set the `HTTPONLY` attribute for Internet Communication Framework (ICF) cookies.

**Reason and Prerequisites**

*Reason:*
Setting the `HTTPONLY` attribute is not implemented in Releases 700, 640, and 620.

*Prerequisites:*
- Import the current ABAP support package.
- Apply the current kernel patch.
- For ABAP Release 620, use the current downward-compatible 640 kernel.

**Solution**
1. Import the current SAP BASIS support package and the current kernel.
2. Set the profile parameter `icf/set_HTTPonly_flag_on_cookies` to activate the `HTTPONLY` attribute for ICF cookies (e.g., for the `MYSAPSSO2` cookie).

**Affected Components**
- **Component:** Basis Components > Middleware > Internet Communication Framework (BC-MID-ICF)
- **Support Package Versions:**
  - SAP_BASIS 620 to 640
  - SAP_BASIS 700

SAP Security Note 2752614: CVE-2019-0319 Content Injection Vulnerability in SAP Gateway

## Overview

**SAP Note Number:** 2752614  
**Title:** [CVE-2019-0319] Content Injection Vulnerability in SAP Gateway  
**Type:** SAP Security Note  
**Priority:** Correction with medium priority  
**Release Date:** June 9, 2020  
**CVSS Score:** 4.3  
**CVSS Vector:** CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:N/I:L/A:N

## Description

The SAP Gateway contains a vulnerability that allows an attacker to inject malicious content, which is then displayed as part of an error message. This can deceive users into believing that the information originates from a legitimate service when it does not.

### Symptom

An attacker can manipulate the SAP Gateway to inject content into error messages, misleading users about the authenticity of the information displayed.

**Example:**

- **Request:**
  ```
  GET ../oDataService/0001/' . Please visit http://www.google.de
  ```

- **Response:**
  ```
  404 - Not Found
  ...
  <message xml:lang="en">Resource not found for the segment ''. Please visit http://www.evilwebsite.com</message>
  ...
  ```

### Reason and Prerequisites

An OData client can craft HTTP requests in a specific manner that allows the backend to correlate error information with the client environment, facilitating content injection.

## Solution

Apply the correction provided in this SAP Security Note to ensure that only OData ABNF syntax is permitted within the URI based on the corresponding OData version. This prevents the injection of unauthorized content.

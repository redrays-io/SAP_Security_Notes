**SAP Security Note 2984627: [CVE-2020-26815] Security Vulnerabilities in SAP Fiori Launchpad NewsTile Application**

**Overview**
This SAP Security Note addresses critical vulnerabilities identified in the SAP Fiori Launchpad's NewsTile Application. Implementing this note is essential to protect your SAP systems from potential exploits.

**Vulnerabilities**

1. **Server-Side Request Forgery (SSRF) Attack**
    - **Description:** Allows unauthorized attackers to send crafted requests to internal systems behind firewalls, potentially accessing sensitive resources.
    - **CVE:** [CVE-2020-26815](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2020-26815)
    - **CVSS Score:** 8.6
    - **Vector:** CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:C/C:H/I:N/A:N

2. **Reflected Cross-Site Scripting (XSS)**
    - **Description:** Enables attackers to inject malicious code into the NewsTile Application, which can be executed in the victim's browser, leading to information disclosure.
    - **CVE:** [CVE-2020-26825](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2020-26825)
    - **CVSS Score:** 6.1
    - **Vector:** CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N

**Impact**
- **SSRF:** Potential access to internal resources, leading to data breaches or further exploitation within the network.
- **XSS:** Information disclosure through execution of malicious scripts in users' browsers.

**Solution**
Apply SAP Security Note 2984627 to mitigate the identified vulnerabilities. The solution restricts the NewsTile Application to use feed URLs only from:
- **Internal Sources:** Same URL and port as the Suite Page Builder application.
- **External CORS-Compliant Sources.**

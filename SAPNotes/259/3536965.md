**SAP Security Note 3536965** addresses several critical vulnerabilities in the Adobe Document Services component of **SAP NetWeaver AS for JAVA**. Below are the key details:

### ğŸ” Description

This security update targets multiple vulnerabilities in Adobe Document Services, which could allow attackers with administrative privileges to exploit the system. The vulnerabilities are detailed below:

#### **CVE-2024-47578**
- **Description:** Server-Side Request Forgery (SSRF) vulnerability allowing attackers to send crafted requests from a vulnerable web application. This can lead to reading or modifying any file and potentially making the entire system unavailable.
- **CVSS:** 9.1  
  **Vector:** CVSS:3.0/AV:N/AC:L/PR:H/UI:N/S:C/C:H/I:H/A:H

#### **CVE-2024-47579**
- **Description:** Allows authenticated attackers to upload or download custom PDF font files, enabling them to read any file on the server without affecting integrity or availability.
- **CVSS:** 6.8  
  **Vector:** CVSS:3.0/AV:N/AC:L/PR:H/UI:N/S:C/C:H/I:N/A:N

#### **CVE-2024-47580**
- **Description:** Enables authenticated attackers to create PDFs with embedded attachments pointing to internal server files, allowing unauthorized reading of any server file without impacting integrity or availability.
- **CVSS:** 6.8  
  **Vector:** CVSS:3.0/AV:N/AC:L/PR:H/UI:N/S:C/C:H/I:N/A:N

**SAP Security Note 1501328 - Potential Information Disclosure Relating to Passwords**

**Header Data**
- **SAP Note/KBA**: 1501328
- **Type**: SAP Security Note
- **Version**: 2
- **Recency**: New
- **Language**: English
- **Category**: Program error
- **Priority**: Correction with high priority
- **Status**: Released for Customer
- **Released On**: December 14, 2010
- **Component**: Basis Components > Web Dynpro > WebDynpro Java

### Description

#### Symptom
A vulnerability allows a malicious user to discover password information in WebDynpro for Java, potentially enabling targeted password attacks.

#### Reason and Prerequisites
Passwords can be exposed in WebDynpro for Java, which can be exploited to further target applications using this framework.

### Solution

**Apply the WebDynpro for Java Patch**

1. **Download Patches:**
   - [Download for SNOTE](https://notesdownloads.sap.com/note/0040000017087912017)
   - [PDF Version](https://userapps.support.sap.com/sap/support/sfm/notes/print/0001501328?language=en-US&token=83CFB040EB6621CC8629E72864945817)

2. **Update via JSPM:**
   - Refer to the [official JSPM documentation](https://me.sap.com/saphelp_nw70/helpdata/EN/1f/c45b4211aac353e10000000a1550b0/frameset.htm) for instructions on updating your system.

3. **Release Specific Instructions:**
   - **NW04 / NW 6.40:**
     - Install SAR file patches and deploy SDAs using JSPM.
   - **NW04S / NW 7.00:**
     - Deploy SCA patches directly using JSPM.
   - **Recommendations:**
     - Apply both `SAPJTECHF.sca` and `SAPJTECHS.sca`. If any archive is not mentioned in the 'SP Patch Level' tab, apply the available patches from the SAP Service Marketplace.

**Support Package Patches**

Ensure you apply the correct support package patches for your system version. You can find the relevant patches [here](https://me.sap.com/sap/support/swdc/notes?cvnr=...).

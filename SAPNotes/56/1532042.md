1532042 - Missing authorization check in SAP Solution Manager

---

**SAP Note**: [1532042](https://me.sap.com/notes/0001532042)

**Type**: SAP Security Note  
**Version**: 4  
**Recency**: Updated  
**Language**: English  
**Title**: Missing authorization check in SAP Solution Manager

### Description

**Symptom:**  
This Security Note has been updated.  
See [Security Note 1604162](https://me.sap.com/notes/1604162) for further details.  
An authenticated user can use certain functions of SAP Solution Manager without having the required authorization. This may result in an escalation of privileges.

**Reason and Prerequisites:**  
SAP Solution Manager does not contain authorization checks for verifying an authenticated user's permissions to access some of its functions. This may lead to undesired system behavior.

### Solution

Import the specified Support Package or implement the [attached correction instructions](https://me.sap.com/corrins/0001532042/212).

### Affected Components

- **Service > SAP Solution Manager > Solution Manager System Landscape (SV-SMG-SYS)**

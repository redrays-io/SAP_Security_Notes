SAP Security Note 1849559: Code Injection Vulnerability in BW-WHM-DST

```markdown
**Overview**

SAP has released Security Note **1849559** addressing a **code injection vulnerability** in the `BW-WHM-DST` component. This vulnerability allows attackers to execute arbitrary program code, potentially leading to system behavior manipulation or privilege escalation without legitimate credentials.

**Vulnerability Details**

- **Description:** The `BW-WHM-DST` component contains code that permits the execution of arbitrary program code chosen by the user. An attacker can leverage this vulnerability to control system behavior or escalate privileges by executing malicious code without having legitimate credentials.
  
- **Impact:** Successful exploitation can lead to:
  - Injection and execution of unauthorized code.
  - Access to additional sensitive information.
  - Modification or deletion of data.
  - Creation of new users with higher privileges.
  - Denial of service attacks.

**CVSS Score**

- **Base Score:** 6.0
- **Vector:** AV:N/AC:M/AU:S/C:P/I:P/A:P

**Affected Software Versions**

The vulnerability impacts the following SAP NetWeaver BW versions:

- **SAP NetWeaver BW 7.00**
- **SAP NetWeaver BW 7.01**
- **SAP NetWeaver BW 7.02**
- **SAP NetWeaver BW 7.11**
- **SAP NetWeaver BW 7.30**
- **SAP NetWeaver BW 7.31**
- **SAP NetWeaver BW 7.40**
- **SAP BW Virtual Component 7.01**

**Solution**

To remediate this vulnerability, apply the relevant **Support Packages** based on your SAP NetWeaver BW version:

- **SAP NetWeaver BW 7.00:** Import Support Package 31 (SAPKW70031)
- **SAP NetWeaver BW 7.01:** Import Support Package 14 (SAPKW70114)
- **SAP NetWeaver BW 7.02:** Import Support Package 14 (SAPKW70214)
- **SAP NetWeaver BW 7.11:** Import Support Package 12 (SAPKW71112)
- **SAP NetWeaver BW 7.30:** Import Support Package 10 (SAPKW73010)
- **SAP NetWeaver BW 7.31:** Import Support Package 9 (SAPKW73109)
- **SAP NetWeaver BW 7.40:** Import Support Package 3 (SAPKW74003)

**Correction Instructions**

Before applying the support packages, ensure you review **SAP Note 875986** for transaction SNOTE to understand the correction instructions.

**Prerequisites**

Ensure that all prerequisite SAP Notes are applied. Refer to the [Prerequisites section](https://me.sap.com/notes/1849559) of the security note for detailed information on required SAP Notes.

For more details and to download the security note, visit the [SAP Security Note 1849559](https://me.sap.com/notes/1849559).
```

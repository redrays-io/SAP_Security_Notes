1488240 - Potential risk of modification of persisted data in EI SAF

```markdown
**SAP Security Note:** 1488240 - Potential risk of modification of persisted data in EI SAF

**Type:** SAP Security Note  
**Version:** 4  
**Recency:** New  
**Language:** English  
**Category:** Program error  
**Priority:** Correction with high priority  
**Status:** Released for Customer  
**Released On:** 14.12.2010  
**Component:** Cross-Application Components > General Application Functions > use CRM-IC (Interaction Center WebClient) > use CRM-BF-SAF (Software Agent Framework)

### Symptom
A malicious user can exploit the EI SAF component and use specially crafted inputs to execute arbitrary database commands to retrieve, modify, or remove data persisted by the system.

### Other Terms
Security, SQL Injection, database

### Reason and Prerequisites
The problem is caused by an SQL injection vulnerability. The code composes an SQL statement including strings that can be altered by a malicious user. The manipulated SQL statement can then be used to retrieve additional information from the database or to potentially modify it.

### Solution
Apply the correction instructions or the corresponding support package.
```

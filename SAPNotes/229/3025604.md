SAP Security Note 3025604: CVE-2021-33664 - XSS Vulnerability in SAP NetWeaver AS ABAP

```markdown
**SAP Security Note 3025604**

**Title:** [CVE-2021-33664] Cross-Site Scripting (XSS) vulnerability within SAP NetWeaver AS ABAP (Applications based on Web Dynpro ABAP)

**Description:**

**Symptom**
Applications based on Web Dynpro ABAP do not sufficiently encode user-controlled inputs, resulting in a Cross-Site Scripting (XSS) vulnerability.

**Other Terms**
- XSS
- Stored XSS
- CSS
- [CVE-2021-33664](https://cve.mitre.org/cgi-bin/cvename.cgi?CVE-2021-33664)

**Solution**
Sufficient output encoding is now performed. Install the Unified Rendering cumulative patch UR2102. See [SAP Note 2090746](https://me.sap.com/notes/2090746) for further instructions.

[SAP Note 2500800](https://me.sap.com/notes/2500800) contains information about cumulative patches for Unified Rendering in general.

**CVSS Score:** 5.4

- **Vector:** CVSS:3.0/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N
- **Details:**
  - **Attack Vector (AV):** Network (N)
  - **Attack Complexity (AC):** Low (L)
  - **Privileges Required (PR):** Low (L)
  - **User Interaction (UI):** Required (R)
  - **Scope (S):** Changed (C)
  - **Confidentiality Impact (C):** Low (L)
  - **Integrity Impact (I):** Low (L)
  - **Availability Impact (A):** None (N)

**Release Date:** June 7, 2021

**Components Affected:**
- **Basis Components > Web Dynpro > Web Dynpro ABAP (BC-WD-ABA)**

**Download Links:**
- [Download for SNOTE](https://notesdownloads.sap.com/note/0040000000819132021)

**Manual Actions:**
You need to perform the manual activity separately in each system where you transport the Note for implementation. Implement the newest Unified Rendering release as advised in [SAP Note 2090746](https://me.sap.com/notes/2090746). At least Unified Rendering release UR2102 is required.
```
